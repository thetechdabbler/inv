# Tech Stack

## Overview
A full-stack TypeScript web application built on Next.js, prioritizing type safety, developer experience, and ecosystem consistency across frontend and backend.

## Languages
TypeScript

Full-stack consistency with a single language across client and server. Type safety catches bugs at compile time rather than runtime, and the ecosystem for web development is unmatched.

## Framework
Next.js

Full-stack framework with SSR/SSG support, file-based routing, API routes, and a massive community. Provides strong conventions for project structure and deployment flexibility.
