# Story: 004-sidebar-redesign

## User Story

**As a** user
**I want** a redesigned sidebar with a futuristic aesthetic
**So that** navigation feels modern, polished, and consistent with the theme system

## Scope

Redesign sidebar with futuristic aesthetic. Backdrop-blur in dark mode, glow border, Lucide icons replacing inline SVG paths, animated active indicator, theme toggle at bottom.

## Acceptance Criteria

- [ ] Sidebar uses backdrop-blur + border glow in dark mode
- [ ] All nav icons are Lucide components
- [ ] Active link has animated left-bar indicator
- [ ] Theme toggle positioned at bottom
- [ ] Mobile drawer transitions smoothly
