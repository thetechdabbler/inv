# Story: 010-insights-redesign

## User Story

**As a** user
**I want** a redesigned AI insights chat with themed message bubbles and quick actions
**So that** I can interact with AI insights in a clear, accessible interface

## Scope

Redesign AI insights chat with themed message bubbles (user vs assistant), typing indicator, shadcn Button quick actions, and an illustrated empty state. Support both light and dark themes.

## Acceptance Criteria

- [ ] User and assistant bubbles are distinctly styled and theme-aware
- [ ] Typing indicator shows animated feedback
- [ ] Quick actions use shadcn Button component
- [ ] Empty state has icon or illustration
- [ ] Page renders correctly in both themes
