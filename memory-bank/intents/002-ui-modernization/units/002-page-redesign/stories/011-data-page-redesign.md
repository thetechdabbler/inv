# Story: 011-data-page-redesign

## User Story

**As a** user
**I want** a redesigned import/export page with styled dropzone and clear feedback
**So that** I can import and export data with a modern, intuitive interface

## Scope

Redesign import/export page with shadcn Card, Alert, Button. Include a styled file dropzone with visual feedback. Support both light and dark themes.

## Acceptance Criteria

- [ ] Export and import sections use shadcn Card component
- [ ] Alerts use shadcn Alert component
- [ ] Buttons use shadcn Button with spinner during operations
- [ ] Dropzone has visual feedback (drag-over state, success/error)
- [ ] Page renders correctly in both themes
