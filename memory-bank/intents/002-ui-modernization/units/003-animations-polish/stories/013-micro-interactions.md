# Micro-Interactions

As a user interacting with the application, I want hover effects, press feedback, staggered card entry animations, and number count-up effects, so that the UI feels responsive, alive, and polished.

## Acceptance Criteria

- [ ] Card grids stagger-in on load with sequential delay
- [ ] Dashboard stat numbers count up from 0 on load
- [ ] Buttons show subtle scale-down on press
- [ ] Cards lift and glow subtly on hover
- [ ] All animations disabled with prefers-reduced-motion
