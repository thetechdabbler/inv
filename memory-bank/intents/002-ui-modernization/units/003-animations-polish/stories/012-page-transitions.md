# Page Transitions

As a user navigating the application, I want page-level fade-in animations when switching routes, so that the experience feels smooth and polished without being distracting.

## Acceptance Criteria

- [ ] Page content fades in on navigation (~300ms)
- [ ] Skeleton loaders transition smoothly to real content
- [ ] Animations disabled when prefers-reduced-motion is set
- [ ] No layout shift during transitions
- [ ] Animations run at 60fps
