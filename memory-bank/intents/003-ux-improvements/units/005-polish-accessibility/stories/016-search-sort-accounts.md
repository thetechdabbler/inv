# Story: Add search and sort to accounts page

## User Story
**As a** user with many investment accounts
**I want** to search by name and sort by different criteria
**So that** I can quickly find the account I'm looking for

## Scope
Add a search input (filters by name in real-time) and a sort dropdown (name, value, type, returns) to the accounts listing page.

## Acceptance Criteria
- [ ] Search input filters accounts by name (case-insensitive)
- [ ] Sort dropdown with options: Name, Value, Type, Returns
- [ ] Search and sort work together
- [ ] Empty search shows all accounts
- [ ] No results state shown when search has no matches
